17// Important terminology to remember:
18// MapBase = The space dedicated to your tile metadata (which graphic each tile is pointing to).
19// There are 2 separate MapBases, for layer 0 (drawn behind) and layer 1 (drawn in front).
20// The default Xist16 assumption is that layer 1 is dedicated to text display.
21// The amount of space required for the MapBase depends on the dimensions of each layer.
22// TileBase = The graphical data for your tile layers.
23// The amount of space required for the TileBase depends on how many tile graphics you have.
24
25// Set the dimensions for your tile layer 0 below and the xist_initialize_tiles() function
26// will take care of configuring all the other values needed by the VERA.
27// Then, in xist_mem.h, make sure to supply enough space for your tile layers. Remember that
28// each tile needs 2 bytes!
