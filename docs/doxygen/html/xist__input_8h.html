Detailed Description

User input functionality.

Contains macros, extern variables, and function signatures for handling user input from the joypad and mouse.

◆ xist_get_joypad()

unsigned short xist_get_joypad ( void )

Returns the current state of the joypad.

Returns the current state of the joypad concatenated into a single 16-bit integer and inverted so that 1 = button is pressed, 0 = button is not pressed. See the macro defines in this file which you can AND the returned short with to check for button presses.

◆ xist_get_mouse_buttons_get_falling_edges()

unsigned char xist_get_mouse_buttons_get_falling_edges ( unsigned char last_state)

Returns which mouse buttons are on a falling edge (have just been pressed and then released)

Returns which mouse buttons that were pressed in the passed-in state are now released (i.e., are on a falling edge). 1 = button is pressed, 0 = button is not pressed. See the macro defines in this file which you can AND the returned short with to check for falling edges.

Parameters
unsignedchar last_state The state of the mouse buttons during the last game tick. Required to tell which buttons were previously pressed and are now released.
Returns
unsigned char
